This is a mod for Remnant 2

# Requirements

- Allow Asset Mods
- SummonableTraits(for traits)


# Usage
- open console
- `SummonHelper <Types|Names>`


# Example

- `SummonHelper gun`
- `SummonHelper gloves`
- `SummonHelper ring`
- `SummonHelper rail`
- `SummonHelper weapon_monorail_c`
- `SummonHelper leech`
- `SummonHelper trait_leech_c`
- `SummonHelper scrap`
- `SummonHelper material_scrap_c`
